## Auto Grader

## Dependencies
- Python Version: 3.6.4

Here's how you can install python 3.6 in ubuntu.
``` sh
sudo apt-get install python3.6
```

## Install
```
git clone <https url>
python3.6 -m venv venv
./venv/bin/activate
pip install -r requirements.txt
```

## Instructions
``` sh
python autograder --github {{username}} list-projects
```
